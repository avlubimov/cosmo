cosmo by mascarenhas

Cosmo is a “safe templates” engine for Lua.

Cosmo allows you to fill nested templates, providing many of the advantages of Turing-complete template engines, without without the downside of allowing arbitrary code in the templates.

Dependencies

Just LPEG.

Install

luarocks install cosmo
